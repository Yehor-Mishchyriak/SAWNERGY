Example dataset: a 1 microsecond production trajectory of the p53 DNA-binding domain.

- Frames: 1000 snapshots saved every 1 ns.
- Files: `p53_DBD.prmtop` (topology), `p53_DBD.pdb` (reference structure), `p53_DBD.nc` (trajectory).
- Intended use: quick-start example for users to learn how to use the package.
